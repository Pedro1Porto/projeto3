# Projeto 3 - Restaurante Responsivo

Este projeto é uma página web moderna para um restaurante, desenvolvida com foco em usabilidade, responsividade e apresentação visual atrativa.

## ✨ Qualidades do projeto

- **Design responsivo**: utiliza estrutura adaptável para oferecer uma ótima experiência em desktop, tablet e mobile.
- **Seções funcionais**: inclui área de cardápio, eventos, carousel e menu responsivo para navegação clara e envolvente.
- **Uso de Bootstrap e JavaScript**: combina componentes dinâmicos com validação e máscaras de formulário, resultando em uma página interativa e profissional.
- **Organização de arquivos**: a estrutura de pastas mantém o projeto limpa e fácil de manter, com CSS e scripts separados de forma lógica.
- **Experiência visual rica**: usa imagens e layouts de destaque para valorizar pratos, eventos e ofertas especiais.

## 📁 Estrutura do projeto

- `index.html` / páginas de conteúdo: marcação semântica para facilitar a navegação e o SEO.
- `main.css` e `style.css`: estilos personalizados para tornar a aparência elegante e única.
- `js/main.js`: interações e comportamento da página.
- `images/`: imagens de cardápio, eventos e slides que enriquecem a apresentação.

## 🛠️ Por que este projeto se destaca

- Excelente como portfólio para web design e desenvolvimento front-end.
- Demonstra domínio de responsividade, navegação móvel e apresentação de conteúdo visual.
- Prova de capacidade de criar interfaces visuais atraentes a partir de um layout organizado.
